data.csv		由第一列至最后一列，依次存储原始测深数据，蓝色波段，绿色波段，红色波段及红外波段反射率（Rrs）。用于模型训练。image.csv		为地形图的光谱反射率信息，原始sentinel-2卫星数据经过大气校正与辐射校正之后，获取不同波段的光谱反射率数据。将这些数据制表，由第一列至最后一列，依次存储蓝色波段，绿色波段，红色波段及红外波段反射率（Rrs）。mask_yd.png		存储着掩膜数据，图中黑色区域为陆地部分，白色区域为水体信息。提示词1.txt		用于输入到大语言模型，绘制密度散点图的提示词。提示词2.txt		用于输入到大语言模型，预测水深地形图的提示词。data.csv Stores the original bathymetry data, blue band, green band, red band, and near-infrared band reflectance (Rrs) sequentially from the first column to the last column. Used for model training.image.csv Contains the spectral reflectance information of the topographic map. Spectral reflectance data for different bands are obtained after atmospheric correction and radiometric correction of the original Sentinel-2 satellite data. These data are tabulated, with blue band, green band, red band, and near-infrared band reflectance (Rrs) stored sequentially from the first column to the last column.mask_yd.png Stores the mask data. The black areas in the image represent land regions, and the white areas represent water information.Prompt word 1.txt Prompt input to the large language model for generating density scatter plots.Prompt word 2.txt Prompt input to the large language model for predicting water depth topographic maps.
